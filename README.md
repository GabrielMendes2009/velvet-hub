# Velvet Hub
O Velvet Hub é um projeto em formato de website planejado como um ponto de encontro definitivo para a franquia de videojogos Shin Megami Tensei: Persona. O objetivo principal da plataforma é centralizar informações relevantes tanto para jogadores veteranos quanto para espectadores, leitores e novos fãs da série.

Dentro do site, os usuários poderão encontrar fatos e curiosidades sobre os bastidores e mecânicas dos vários jogos da franquia, Perfis de Personagens onde haverá informações sobre os protagonistas e integrantes principais de cada título, Visão Geral da História e um Guia de Plataformas que consiste em um mapeamento completo de onde e em quais formatos as obras estão disponíveis para o consumo.
